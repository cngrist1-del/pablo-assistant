## 7. Tools & Scripts for Automation (Bonus)

Below is a minimal **Python script** (uses only standard libraries + `pytube` if needed) that can auto‑generate a simple slideshow from a folder of images and an audio file. This is optional – you can do everything manually, but the script speeds up production.

```python
#!/usr/bin/env python3
# generate_rain_video.py
import os
import subprocess
import sys

# ---- CONFIGURATION ----
INPUT_DIR = "assets"                # Contains images: img_001.jpg, img_002.jpg ...
AUDIO_FILE = "assets/rain_loop.mp3"
OUTPUT_FILE = "output/rain_video.mp4"
IMAGE_DURATION = 5                  # seconds per slide
OUTPUT_RES = "1920x1080"
FRAME_RATE = 30

# ---- CHECKS ----
if not os.path.isdir(INPUT_DIR):
    print(f"Error: {INPUT_DIR} not found. Create it and add images.")
    sys.exit(1)
if not os.path.isfile(AUDIO_FILE):
    print(f"Error: {AUDIO_FILE} not found.")
    sys.exit(1)

os.makedirs("output", exist_ok=True)

# ---- GENERATE FILELIST.TXT FOR FFMPEG ----
images = sorted([f for f in os.listdir(INPUT_DIR) if f.lower().endswith(('.png', '.jpg', '.jpeg'))])
with open("filelist.txt", "w") as f:
    for img in images:
        # each image loops for IMAGE_DURATION seconds
        f.write(f"file '{os.path.join(INPUT_DIR, img)}'\n")
        f.write(f"duration {IMAGE_DURATION}\n")

# ---- RUN FFMPEG TO BUILD VIDEO ----
# (Assumes ffmpeg is installed – available on most Linux/macOS systems)
cmd = [
    "ffmpeg",
    "-y",                               # overwrite output
    "-f", "concat",
    "-safe", "0",
    "-i", "filelist.txt",
    "-i", AUDIO_FILE,
    "-vf", f"scale={OUTPUT_RES}:force_original_aspect_ratio=decrease,pad={OUTPUT_RES}:(ow-iw)/2:(oh-ih)/2",
    "-r", str(FRAME_RATE),
    "-c:v", "libx264",
    "-preset", "fast",
    "-crf", "23",
    "-c:a", "aac",
    "-shortest",
    OUTPUT_FILE
]

print("Running ffmpeg... this may take a few minutes.")
subprocess.run(cmd, check=True)
print(f"Video saved to {OUTPUT_FILE}")
```

> **How to use:**
> 1. Install ffmpeg (`sudo apt install ffmpeg` or `brew install ffmpeg`).
> 2. Place images (1920×1080) in `assets/` folder.
> 3. Put a rain MP3 file as `assets/rain_loop.mp3`.
> 4. Run `python3 generate_rain_video.py`.
> 5. Result appears in `output/rain_video.mp4`.

*Optional: Use `pytube` to automatically upload the video, but you will need to handle API keys and OAuth – we can set that up later.*
